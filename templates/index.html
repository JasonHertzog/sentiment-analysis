Training Information

This sentiment analysis model was trained using the IMDb Movie Reviews Dataset. The dataset consists of almost 50,000 movie reviews labeled as positive or negative.

I also used the sentiment140 data set to see if better generalizations could be made given much more data. It contains 1.6M tweets that were annotated as negative, neutral, or positive. For our use we only care about positive and negative. Sentiment140 data set.

The model was built using the Logistic Regression method, a type of regression analysis used for predicting the outcome of a categorical dependent variable based on one or more predictor variables. Learn more about Logistic Regression on Wikipedia.
